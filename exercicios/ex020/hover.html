Lembrando, pseudo hover, serve para mostrar uma interação ao passar o mause em cima, do elemnto q ela estar ativa.

Para isso devo criar a tag div e dentro dela botar uma paragrafo q desejo esconder, dai no meu css crio div > p q quer dizer q minha div e pai do p, display none

E depois div:hover > p, display block, assim q faço uma interação como a cima
