Aqui estou usando tudo que aprendi sobre formularios.

Enfim são muitos conteudos para criar formularios, lembrando que a maioria deles estão na tag input.

so no caso da texarea que usei para digitar uma mensagem, que é uma tag, do mesmo nome.

tamém tem muitas variaçoes de configurações.

  • Como required, para tornar uma capo de preenchimento obrigatorio.

  • min e max, para definir uma quantidade, selecionavel

  • maxlength, para definir uma quantidade maxima, de digitos ou letras q podem ser digitada pelo usuario

  • placeholder, para criar, uma mensagem de informção, dentro do conteudo
