Exemplo de caixa box-level

Paragrafos também sao caixas box-level, mas o links sao caixas de box-inline level vamos ver como tudo isso fuciona.

LEMBRANDO: as tags padding serve para abrir espaço dentro de um caixa, ou quadrado criado pelo div ou nav, right = direita bottom = baixo padding-top = cima padding left = esqueda

LEMBRANDO: Width = medida horizontal, e height = medida da altura., configurando ela com px para chegar a um resulatado q eu quero.

LEMBRANDO: a tag border serve para criar bordas em elementos, exemplo, border style:solid = criar uma traçado q forma uma caixa, border color para a cor, e em border - style também tenho uma variasoes, como dashed, que faz minha borda ficar com traçasdos, dotted para minha borda ficar com pontinhos, double q faz ficar com linha dupla, e grove, q dar um pequeno efeito 3d em minha borda.

E tambem tem a tag outline, color style widith que serve para criar outra borda por fora da primeira borda, podendo escolher cor com color, e expressura com widith px, e o estilo com styly.

Lembrando que tag margin serve para regular as posiçao q eu quero deixar minha caixa na tela.

dendo elas margin = top px para cima, margin - right = direita, margin - bottom = baixo, margin left = esquedamargin auto para centralizar a caixa, lembrando q devo sempre centralizar ela com margin auto, e sempre usar as direiçoes q quero regular-la começando de cima, e com sentido horario regulando as outras, o os px da quantidade q precisso.

OBS: shorthand sao uma simplificação de valores, exemplo margin, eu psso indicar todas as medidas em uma unica linha com margin e as medidas em px em seguidas.

EXEMPLO: AO INVES DE EU BOTAR MARGIN E CADA POSIÇÃO, POSSO SIMPLESMENTE USAR TAG MARGIN - 10px, 10 px 10px 10px ou como neste caso sao px repetidos, posso usar simplesmente px10 uma unica vez, q terei o mesmo resultado. lembrando q sempre conto os px com sentido horario começando de cima.
